# Student Management System

## 📌 About the Project

A Student Management System built using Laravel and MySQL to manage student records efficiently. The project includes features for adding, updating, viewing, and deleting student information.

---

## 🚀 Features

* Add new students
* Edit student details
* Delete student records
* View student list
* Form validation

---

## 🛠️ Technologies Used

* Laravel
* PHP
* MySQL
* Blade Templates
* Bootstrap/CSS

---

## 🎯 What I Learned

* Laravel MVC structure
* Routing and Controllers
* CRUD operations
* Database migrations
* Blade templating
